What is Jobs Common?

We created this project to make it easy for PHP developers to access job board APIs.

While there are tons of job boards that will give you access to their data, every one of them returns that data in its own special way, making it hard to compare, store, or read data from lots of job boards. We figured, "Why not give devs one simple set of clients to access job board data?" And here we are.

In order to use this project, you just include the providers you want to use via Composer, and call the "getJobs()" method. Each provider will give you a standard schema.ort JobPosting object, making it easy to compare results across job boards, store the data, or do your own analysis.

Providers

Each job board supported has a provider that extends the Job Common package's AbstractProvider, and implements the required abstract methods. We currently provide official support for 13 job board APIs.

Click on the name of a provider below to access examples and documentation.

Installation

Via Composer

$ composer require jobbrander/jobs-common

How to Use

Usage varies by provider, but the basic pattern is:

$provider = new Provider([
    'required_param' => XXXX,
]);
$jobs = $provider->setKeyword('search keyword')->getJobs();

Specific documentation is available for each provider.

Technical Details

This package is compliant with PSR-1, PSR-2, PSR-4, and PSR-7. We use Guzzle as an HTTP Client.

The following versions of PHP are supported:

  • PHP 5.5
  • PHP 5.6
  • PHP 7.0
  • HHVM
