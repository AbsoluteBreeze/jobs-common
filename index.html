What is Jobs Common?

A few months ago I was playing around with several job board APIs when I realized that there was a huge lack of standardization in the way in which their APIs were written and accessed. Some of the problems I spotted right out of the gate were:

  • Every job board’s API demands different request parameters
  • Every API returns a different object type; some will only do XML, some JSON, and the names of fields can be just about anything
  • Some require API/Publisher keys
  • Some have libraries, some don’t; very few have good ones in PHP
  • Even if there are libraries, they still return non-standard results so you can’t compare one with another
  • Many of the PHP libraries are outdated, and don’t take advantage of stuff like Composer which makes it super easy to include in a project

So, I created an early version of this project and blogged about it back in March, 2015. Steven Maguire saw the post and asked if I was interested in collaborating on a more mature version. Thanks to his well-crafted rewrite, we have what is here today.

Even though the code has improved significantly, the goal has remained the same: to provide PHP developers with a simple and standardized way to access job board APIs.
