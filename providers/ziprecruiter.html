Ziprecruiter Jobs PHP API Client

Use the Jobs Common Ziprecruiter Jobs Provider to connect your PHP application with the Ziprecruiter job board API.

$client = new JobBrander\Jobs\Client\Providers\Ziprecruiter([ 'api_key' => 'YOUR ZIPRECRUITER API KEY' ]); $jobs = $client->setKeyword('project manager') ->setLocation('Chicago, IL') ->getJobs();
