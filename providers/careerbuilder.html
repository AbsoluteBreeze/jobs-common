Careerbuilder PHP API Client

Github | Packagist

Use the Jobs Common Careerbuilder Provider to connect your PHP application with the Careerbuilder job board API.

Installation

Use composer:

composer require jobbrander/jobs-careerbuilder

Usage

Get the latest project manager jobs in Chicago: 

$client = new JobBrander\Jobs\Client\Provider\Careerbuilder([
    'DeveloperKey' => 'YOUR CAREERBUILDER DEV KEY',
]);

$jobs = $client->setKeyword('project manager')
    ->setCity('Chicago')
    ->getJobs();

For more detailed usage and documentation, see this project's readme
